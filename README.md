# Popular Movies

Popular movies app for Udacity's Android Developer Nanodegree

### Screenshots

![](https://raw.githubusercontent.com/alexsanderfr/popular-movies/master/images/1.png)

### Prerequisites

```
Android Studio
```

### Installing

```
Clone this repo
Open with Android Studio
Run in device or emulator
```
